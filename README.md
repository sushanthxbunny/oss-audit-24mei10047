## Project Description
This project analyzes Python as an open-source software, including its origin, licensing, Linux footprint, ecosystem, and comparison with proprietary software.  
It also includes five shell scripts demonstrating Linux and automation concepts.

---

## Scripts Overview

### Script 1 — System Identity Report
Displays system details such as kernel version, user, uptime, and OS information.

### Script 2 — FOSS Package Inspector
Checks if Python is installed and displays package details.

### Script 3 — Disk and Permission Auditor
Analyzes directories and displays permissions and sizes.

### Script 4 — Log File Analyzer
Counts keyword occurrences in log files.

### Script 5 — Manifest Generator
Generates a personalized open-source manifesto.
